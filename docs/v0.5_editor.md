# 📝 Pyqttyai v0.5 — Script Editor Features & Shortcuts

> Companion to the [v0.5 README section](../README.md#v05--pyqttyai-v05--voice--nlp-intelligence-layer).

The Script Editor sits at the top of every Device Tab. In v0.5 it became a
**first-class CLI authoring environment** — keyboard-driven, regex-aware,
and tightly integrated with the NLP rules engine.

## 📐 Configurable Indentation

v0.5 introduces a **global indent setting** (1–8 spaces) shared by every
Script Editor in the workspace.

- ⚙️ Configure via: **Settings → 📐 Script Editor Indentation → N spaces**
- 🎯 **Default: 1 space** — Cisco IOS style (most configs are 1-space indented)
- 📊 Active value is shown permanently in the status bar (`📐 1 space`)
- 💾 Persisted via `QSettings("Pyqttyai", "Pyqttyai")` — survives restarts
- ➡️⬅️ `Tab` / `Shift+Tab` apply this unit to the selection (multi-line aware),
  preserving the cursor logically

## 🔢 Separator Normalization (the Hidden Gem)

One of v0.5's quiet productivity wins: turn **anything** you typed (or
dictated via voice) into a properly-formatted IP, IPv6, or MAC.

### How it works

1. Place the cursor inside a "hex-ish" token (digits, hex letters, separators).
2. Press the desired separator shortcut.
3. The editor identifies the surrounding token boundaries automatically
   (no need to select) and rewrites it.

You can also **select a range first** for explicit control.

### Examples (from the demo video)

| Before | Shortcut | After |
|--------|----------|-------|
| `192 168 1 1` | `Ctrl+.` | `192.168.1.1` |
| `192,168;1.1` | `Ctrl+.` | `192.168.1.1` |
| `fe80 0 0 0 1` | `Ctrl+;` | `fe80::1` |
| `192.168.10.acad.cafe.2.2.3` | `Ctrl+;` | `192:168:10:acad:cafe:2:2:3` |
| `aa bb cc dd ee ff` | `Ctrl+-` | `aa-bb-cc-dd-ee-ff` |
| `00:11:22:33:44:55` | `Ctrl+Shift+Space` | `00 11 22 33 44 55` |

> 💡 **Pairs beautifully with voice input.** Say *"one ninety-two dot one
> sixty-eight dot one dot one"*, let the NLP rule produce digits, then hit
> `Ctrl+.` to clean it up. As shown in the demo, even a typo like missing a
> dot can be fixed with one keystroke.

## ✂️ Smart Cut/Copy Without Selection

When **nothing is selected**, `Ctrl+C` and `Ctrl+X` operate on the **whole
current line** (including the trailing `\n`). This matches VS Code's
convention and is *way* faster than triple-clicking:

- 📋 Paste somewhere else → the line lands on its own line.
- ♻️ Cut + paste a few lines down → instant line move.
- 🚫 With a selection, behavior is **standard** — your existing muscle
  memory is untouched.

## 🎯 `Ctrl+Shift+A` — Apply NLP Rules In-Place

The single most powerful editor shortcut in v0.5.

### Workflow (from the demo)

You spoke the initial Cisco config, but the password rule failed because
`class` has fewer than 10 characters. Fix the line manually:

```
username Cisco password class1234567
```

Then press `Ctrl+Shift+A` — the line is re-fed through the NLP rules engine
and replaced in-place with the corrected config block. No need to re-speak
the entire sentence. 🎯

### Use cases

- 📝 Type sloppy natural-language intent: `the name router1 has password firefox`
- 🎯 Press `Ctrl+Shift+A` → editor replaces it with:
  ```
  hostname ROUTER1
  enable secret @firefox!
  ```
- 🔁 Iterate: write 10 lines of intent, then go line-by-line applying rules.

## 🔎 Find & Replace Bar (`Ctrl+Shift+F`)

A VS Code-style find bar lives **above** each Script Editor and TTY Console.

### Features

| Toggle | Meaning |
|--------|---------|
| `Aa` | Case sensitive |
| `⟦ab⟧` | Whole word |
| `.*` | Regex mode |
| `⊂` | Find in current selection only |

### Navigation

- `Enter` in find box → **next** match
- `Shift+Enter` → **previous** match
- `F3` / `Shift+F3` → next / previous (works even when the bar is closed)
- `Esc` → close the bar

### Replace

- 🔁 Single replace (`⌌`)
- 🔁🔁 Replace All (`⇄`)

### Real-world example (from the demo)

The NLP rule produced `name Cisco` instead of `Cisco`. Fix:

1. `Ctrl+Shift+F`
2. Find: `name Cisco`
3. Replace: `Cisco`
4. Click `⇄ Replace All` → 8 occurrences replaced. ✨

> 💡 The match counter shows `3 of 17` so you always know where you are.

## 🖥️ Bonus — TTY Console Find Bar

The same Find Bar component is wired into the **TTY Console** at the bottom
of every Device Tab — but with a deliberate UX twist:

- `Ctrl+Shift+F` opens the bar **only when text is selected** (to avoid
  accidentally interrupting your typing into a live router session).
- No Replace section — the TTY is read-only by design.

## 🧠 Behavioral Details

- ⏮️ **Undo/Redo preserved** across all v0.5 features (indent, dedent,
  normalize, apply-rule, find-replace) — each is wrapped in a single
  `beginEditBlock` / `endEditBlock`.
- 📍 **Cursor preservation** — indent/dedent and normalization restore the
  cursor logically (shifted by inserted/removed characters), not physically.
- 🛡️ **Bad regex won't crash** — invalid patterns in Find or NLP rules turn
  the input box red with a tooltip showing the error.
- 🎯 **Widget-scoped shortcuts** — editor shortcuts only fire when the
  editor has focus, so `Ctrl+;` (IPv6) won't conflict with the TTY console.

## 💡 Power-User Recipes

### 📋 Convert a paragraph of intent into config

```
1. Paste/dictate intent (multi-line natural language)
2. For each line:
     - Park cursor on the line
     - Ctrl+Shift+A  (apply rules)
3. Select All → ▼ Send Script
```

### 🔢 Clean up addresses pasted from an email

```
1. Paste raw text
2. Ctrl+Shift+F → enable regex → find: \d+[\s,;.\-]+\d+[\s,;.\-]+\d+[\s,;.\-]+\d+
3. F3 to walk through each match
4. Ctrl+.  on each one to normalize
```

### ✂️ Move 5 lines up

```
1. Select 5 lines
2. Ctrl+X
3. Move cursor up
4. Ctrl+V
```

### 🔁 Toggle between IPv4 and IPv6 view (from the demo)

If you have:
```
192.168.10.acad.cafe.2.2.3
```
Copy it, press `Ctrl+;` to view it as `192:168:10:acad:cafe:2:2:3`. Add two
colons (`::`) where you want zero compression, and you're set. The reverse
works too — `Ctrl+.` converts back.
